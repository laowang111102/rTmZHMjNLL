# Springboot房屋租赁系统简介 python1204

## 项目概述


## 技术栈

- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK 1.8、Mysql、Maven、Navicat
- 技术栈：Java、SpringBoot、Mybatis、Html、Css、Vue、Mysql

## 功能模块

- 人中心
- 房屋类型管理
- 房屋信息管理
- 预约看房管理
- 合同信息管理
- 房屋报修管理
- 房屋评价管理
- 我要当房东管理
- 留言板管理
- 系统管理

## 系统角色

- 用户：普通用户、房东
- 管理员：负责系统管理、内容审核等

## 运行环境

- 服务器：建议使用支持Java和MySQL的Linux服务器
- 客户端：任何支持现代Web浏览器的设备

## 部署步骤

1. 准备Java 1.8以上版本运行环境
2. 准备MySQL数据库
3. 部署后端代码（Springboot应用）
4. 部署前端代码
5. 配置服务器，启动应用程序

## 目录结构

```
├── backend                # 后端代码目录
├── frontend               # 前端代码目录
└── sql                    # SQL文件目录
    ├── schema.sql         # 数据库结构文件
    └── data.sql           # 初始化数据文件
```
